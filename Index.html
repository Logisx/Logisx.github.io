Grid layout:


A grid is a set of intersecting horizontal and vertical lines defining columns and rows.

Elements can be placed onto the grid within these column and row lines.

Reaction:


You can organise information with the use of accurate measurement units(ex. "px") or with relative units of measurement(ex. "%").

Relative units will adjust to the screen size.

Settings:


  • "display: grid;" makes your class a grid layout.

  • You arrange lines of grid with "grid-template-columns:" and "grid-template-rows:".

  • "grid-gap:" arranges the size of gap between blocks.

  • You can organise the position of your blocks with "grid-template-areas:".

  • Or you can define it for each block with "grid-column/row-start/end:".

Features:


  • Flexible Track Sizes
    You can use the fr unit (Fraction Unit) to assign any specified pixel value to the grid.
  • Item Placement
    CSS grid has made it much easier to position items in the container.
